为了让广大开发者可以定制适合不同业务、不同环境的FIS，我们将FIS设计成了一个插件系统。FIS在编译文件时，通过调用不同的插件来对文件进行处理，针对不同的编译过程以及内容，开发者可以配置不同的插件来进行处理。同时，你如果需要为一个团队或某类业务制定一套前端最佳实践方案，也可以通过配置FIS的方式来创造属于你的FIS。
