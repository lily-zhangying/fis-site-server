F.I.S简介

  • 什么是F.I.S

快速上手

  • 安装
    • npm install -g fis
  • 三条命令，满足你的所有开发需求：
    • fis install <name>
    • fis release [options]
    • fis server <command> [options]

三种语言能力

  • 关于三种语言能力
  • 在html中：定位资源、嵌入资源、声明依赖
  • 在js中：定位资源、嵌入资源、声明依赖
  • 在css中：定位资源、嵌入资源、声明依赖
