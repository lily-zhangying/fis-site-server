示例安装

我们使用FIS-Plus解决方案进行FIS官网开发，用户可直接下载安装示例来直观的观察体验FIS-Plus解决方案的开发模式：

//将官网项目拆分为2个模块进行开发
$ fisp install pc-demo-common
$ fisp install pc-demo-home

或者你可以直接安装整个demo示例

$ fisp install pc-demo

为了可以正常运行demo，需要安装FIS本地的调试预览环境

$ fisp server install pc

示例预览

在你执行命令下的目录下会下载安装common和home两个模块，具体目录如下：

├── common   //公用资源模块
│   ├── fis-conf.js  //配置文件
│   ├── page     //页面模板目录
│   ├── plugin   //smarty plugin目录
│   ├── static   //静态资源目录
│   └── widget   //模块化组件目录
└── home    //业务模块
    ├── fis-conf.js
    ├── page
    ├── static
    ├── test    //测试数据目录
    └── widget

用户根据目录规范，将代码进行分类归置，具体的开发规范可查看解决方案规范。接下来我们将模块进行发布，在本地环境中预览页面：

//启动本地server服务
fisp server start
//在common模块路径下发布common模块
fisp release -cw
//在home模块路径下发布home模块
fisp release -cw
//在浏览器中访问页面，可预览FIS官网
http://localhost:8080/home/page/index

发布完模块后，用户则可以在浏览器中访问编译后的页面，此时你就看到一个由FIS-Plus解决方案生成的页面，你可以尝试修改模块，感受下FIS-Plus的神奇魅力.
